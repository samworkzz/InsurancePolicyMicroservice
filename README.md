<h1>Insurance Policy Notes Microservice</h1>

A lightweight, RESTful microservice built with .NET 8 Minimal APIs designed to manage internal notes for customer insurance policies. This project demonstrates clean architecture principles using the Repository Pattern and includes a comprehensive test suite with xUnit.
<h2>Features</h2>

Minimal API Architecture: Low-overhead, high-performance API endpoints.

Repository & Service Layers: Decoupled business logic and data access.

In-Memory Database: Uses EF Core InMemory for fast prototyping and testing.

Comprehensive Testing:

Unit Tests: Isolated tests using Moq for service logic.

Integration Tests: End-to-end API tests using WebApplicationFactory.

Swagger/OpenAPI: Built-in API documentation and testing interface.

<h2>Tech Stack</h2>

Framework: .NET 8.0

Database: Entity Framework Core (In-Memory)

Testing: xUnit, Moq, Microsoft.AspNetCore.Mvc.Testing

Documentation: Swashbuckle (Swagger UI)
